# study_android-playground 独立启动说明

## 1. 前置环境

- Node.js >= 18.19.0
- 建议使用 `corepack` 管理 pnpm
- Android 设备已通过 adb 连接（`adb devices` 可看到设备）

## 2. 安装依赖

在项目根目录执行：

```bash
cd /Users/yaohongliang/work/liuyao/AI/UI/study_android-playground
corepack pnpm install
```

## 3. 启动前端页面

```bash
cd /Users/yaohongliang/work/liuyao/AI/UI/study_android-playground
corepack pnpm run dev:ui
```

默认访问：

- http://localhost:3000

## 4. 启动 Android Playground 服务端

```bash
cd /Users/yaohongliang/work/liuyao/AI/UI/study_android-playground
corepack pnpm run dev:server
```

说明：

- 会自动扫描并选择已连接设备
- 若默认端口被占用，会自动切换到可用端口（如 `5801` / `5701`）

## 5. 常见问题

### 5.1 `pnpm: command not found`

使用 `corepack pnpm` 代替 `pnpm`：

```bash
corepack pnpm --version
```

### 5.2 端口占用

日志会提示自动换端口，按日志中的实际地址访问即可。

### 5.3 没有设备

请检查：

- USB 调试已打开
- 设备已授权电脑调试
- `adb devices` 显示状态为 `device`
