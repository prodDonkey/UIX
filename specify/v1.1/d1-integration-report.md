## 4. 一致性结论
- 状态一致性：`queued -> running -> success/failed/cancelled` 与数据库一致。
- 步骤一致性：`current_task/current_action/progress_json` 与右侧实时画面对应。
- 日志一致性：实时日志持续追加，可对照当前执行阶段。
- 报告一致性：终态后可按既有链路下载/预览报告。
